# Q-SpecTrum Skills Index / 技能索引

> **For Users**: Say any trigger phrase below to activate a skill in chat
> **For AI**: Reference this file when user asks about available skills
> **Version**: 1.0 (2026-04-18)

---

## How to Use Skills / 如何使用技能

In any Q-SpecTrum chat session, simply say:
- **"Show me available skills"** — displays this catalog
- **"I want to use [skill name]"** — activates a specific skill
- **"[trigger phrase]"** — directly triggers the skill

在 Q-SpecTrum 對話中，直接說出觸發短語即可啟動技能。
