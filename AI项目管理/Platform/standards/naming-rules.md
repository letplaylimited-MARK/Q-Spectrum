# Naming Rules

## Entity codes

- Project: `PRJ-0001`
- Pain point: `PAIN-0001`
- Demand: `DEM-0001`
- Demand item: `DI-0001`
- Solution: `SOL-0001`
- Agent: `AGT-0001`
- Workflow: `WF-0001`
- Handoff: `HOF-0001`
- Validation: `VAL-0001`
- Defect: `BUG-0001`
- Document: `DOC-0001`

## Folder naming

- Use ASCII for technical platform folders
- Use kebab-case for generated technical file names
- Keep existing Chinese knowledge files unchanged unless there is a migration reason
