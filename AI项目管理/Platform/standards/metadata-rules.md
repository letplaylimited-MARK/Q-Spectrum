# Metadata Rules

## Required fields for linked documents

- `title`
- `document_type`
- `file_path`
- `content_hash`
- `status`
- `linked_entity_type`
- `linked_entity_code`

## Required fields for operational YAML or JSON imports

- `source_system`
- `schema_version`
- `created_at`
- `owner`
- `status`
