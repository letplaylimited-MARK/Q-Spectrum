## 标准工作流

### 技能开发全流程

```
需求 → Skill 00(路由) → Skill 02(SOP) → Skill 03(选型) → Skill 04(规划) → Skill 05(验收) → 交付
```

### 快速流程

```
简单需求 → Skill 00(路由) → Skill 03/05(按需) → 完成
```
