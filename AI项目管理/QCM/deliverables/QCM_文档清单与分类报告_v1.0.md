## 一、文档分类总览

### 分类标准

| 分类 | 定义 | 维护要求 |
|------|------|---------|
| 🔴 **核心文档** | 对外发布的必要资产 | 持续维护，版本同步 |
| 🟡 **可选文档** | 辅助理解，但不紧急 | 定期审查 |
| 🟢 **可归档** | 历史版本或内部参考 | 归档，不再主动维护 |

## 五、版本同步规则

### 同步要求

核心文档之间需要保持版本同步：

| 文档组 | 同步要求 | 负责人 |
|--------|---------|--------|
| 主论文 (MD/DOCX/PDF) | 同步更新 | 论文负责人 |
| 幽灵通道论文 (MD/DOCX/PDF) | 同步更新 | 论文负责人 |
| 白皮书 (技术版/高管版) | 同步更新 | 白皮书负责人 |
| RFC (v0.1/v0.2/v1.0) | 递增版本 | 协议负责人 |
| SDK (Python/TypeScript) | 同步功能 | SDK负责人 |

### 版本命名规则

```
主论文: v11.1 (当前) → v11.2 (下一版)
幽灵通道论文: v1.0 → v1.1
技术白皮书: v2.0 → v2.1
协议规范: v1.0-SDK草案 → v1.0
RFC: v0.2 → v0.3 → v1.0
SDK: v0.1.0 → v0.2.0 → v1.0.0
```
