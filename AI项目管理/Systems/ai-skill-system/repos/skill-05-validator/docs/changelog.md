## v1.0.0 — 2025年（首发版本）

### 首次发布

**新增**：
- 5阶段工作流（材料接收→五维度验收→缺陷分级→发布决策→报告输出）
- 五维度量化评分（功能完整性40%/文档完整性25%/可执行性20%/接口规范性10%/安全性5%）
- P0-P3 四级缺陷分级（每级有明确定义和发布影响）
- 发布决策三档（PASS/CONDITIONAL_PASS/FAIL，数学条件明确）
- 飞轮机制（缺陷模式归档→上游Skill改进建议）
- 历史缺陷优先复验规则
- 版本升级规则（P0修复→MINOR+1，P1修复→PATCH+1）
- 修复路由规则（缺陷类型→对应修复Skill）

**技术规格**：
- SKILL.md description：424字符（合规，≤1024）
- 触发词：中文7个，英文6个
- 工程包文件数：10个（v1.0.1后：11个）
