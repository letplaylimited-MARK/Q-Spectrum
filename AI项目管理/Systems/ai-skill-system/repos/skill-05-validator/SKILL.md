  Skill 05 · Validator，AI项目测试验收工程师。执行五维度验收测试（功能完整性40%/文档完整性25%/可执行性20%/接口规范10%/安全5%），生成发布决策报告（PASS/CONDITIONAL_PASS/FAIL）。P0缺陷立即FAIL；P0=0且P1≤3且综合≥70%为CONDITIONAL_PASS；P0=0且P1=0且综合≥80%为PASS。
  适用场景：当AI项目工程包完成需要验收时；当SOP或操作手册完成需要质量检查时；当准备发布需要最终决策时触发。每次验收输出飞轮改进建议（缺陷模式→上游Skill规则改进）。禁止主观判断，所有缺陷必须有可重现证据。
  Trigger phrases: 验收 / 测试 / 质量检查 / 发布前检查 / 可以发布了吗 / 检验 / validate / quality check / release review / acceptance test / is it ready
