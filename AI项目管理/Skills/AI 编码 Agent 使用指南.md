# Skill 07 · AI 编码 Agent 使用指南

## 🎯 激活词
- `Agent 编码`
- `Cursor 使用`
- `OpenCode 指南`

## ⏱️ 使用时机
当你需要使用 AI 辅助编程工具（如 Cursor, OpenCode, Aider）进行开发，但缺乏最佳实践或环境配置指南时触发。

## 🛠️ 核心工作流
1. **工具选型**: 根据项目类型和需求，推荐最适合的 AI 编码工具。
2. **环境配置**: 提供一键配置脚本或详细的环境搭建步骤。
3. **工作流集成**: 展示如何将 AI Agent 融入现有的 Git 分支管理和 CI/CD 流程。
4. **提示词库**: 提供针对代码生成、重构、Debug 的专用提示词模板。

## 📦 交付物
- 🛠️ 环境配置清单
- 💻 编码工作流 SOP
- 📝 专用提示词模板

## ⚠️ 常见错误提醒
- ❌ 盲目接受代码 -> ✅ 必须人工 Review AI 生成的代码。
- ❌ 忽略上下文 -> ✅ 确保 Agent 能访问到完整的项目上下文。
