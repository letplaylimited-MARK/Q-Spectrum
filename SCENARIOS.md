# Q-SpecTrum Scenario Journeys / 場景旅程

> **For Users**: Say "start [scenario name] scenario" to begin a guided journey
> **For AI**: Guide the user through each step, switching roles as indicated
> **Version**: 1.0 (2026-04-18) | **Total**: 12 scenarios, 68 steps

---

## How to Use Scenarios / 如何使用場景

Say any of these to start:
- **"Start the e-commerce scenario"** — begins the 8-step e-commerce journey
- **"I want to build a startup MVP"** — begins the 7-step MVP validation journey
- **"Help me with content creation"** — begins the 6-step content studio journey

The AI will guide you step by step, switching to the appropriate role at each stage. You can pause, skip steps, or go back at any time.

在對話中說出場景名稱即可開始引導式旅程。AI 會在每一步自動切換到合適的角色。
