# BRAIN-KB Limitations

> AI 能力邊界認知。每次執行前自我提醒。
> 誠實條款的實作基礎。

## 已知限制

### L1: 上下文長度限制
- 無法一次讀取整個專案
- 需要分段閱讀、摘要、優先級排序

### L2: 自我評估偏誤
- 同一模型無法公正評估自己的輸出
- 開發軌與評審軌必須用不同視角/角色

### L3: 沙盤推演 ≠ 真實分散式協作
- 3 視角沙盤只是觀點多樣性
- 不是真正的多 Agent 協作
- 置信度永遠要標注：高/中/低

### L4: Hallucination 風險
- 永遠存在
- 防禦：不確定的不說、有來源才引用、每一步驗證
- 無法完全消除，只能管理

### L5: 無法執行真實外部操作
- 無法直接安裝軟體、啟動服務、監聽 port
- 所有操作透過 Bash 工具間接執行
- 需要考慮權限、路徑、編碼等問題
