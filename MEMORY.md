## 6. 知识共振公式

```
R = 0.35×K_sim + 0.25×C_comp + 0.25×I_freq - 0.15×E_div

K_sim: 知识相似度
C_comp: 互补性
I_freq: 交互频率
E_div: 熵散度

R > 0.85 → 涌现
0.5 ≤ R ≤ 0.75 → 需验证
R < 0.5 → 需补充
```
